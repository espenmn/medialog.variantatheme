medialog.variantatheme Installation
------------------------------------

To install medialog.variantatheme into the global Python environment (or a workingenv),
using a traditional Zope 2 instance, you can do this:


* Add ``medialog.variantatheme`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        medialog.variantatheme
